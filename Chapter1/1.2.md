### 1.2 [1] Can a higher-level programming language instruct a computer to compute more than a lower-level programming language?

Programming languages generally intended to instruct
some kind of abstratc computer what to do, so if it is able to  specify all actions
for some universal computitional machine,
then, it is possible to translate anything described by it
to actual computer's specific low-level programming language.

But in case if particular programming language to solve
some particular type of problems, then it could be impossible.
