### 2.44 What operation(s) can be used to convert the binary representation for 3 (i.e., 0000 0011) into the ASCII representation for 3 (i.e., 0011 0011)? What about the binary 4 into the ASCII 4? What about any digit?

First of all let's note that ASCII digits codes confirm to formula:

ASCIICode(digit + 1) = ASCIICode(digit) + 0000 0001

so we have:

ASCIICode(digit) = ASCIICode(0) + digit * 0000 0001

Now we have ASCIICode(0) = x30 = 00110000

So each subsequent digit is simply addition of 0000 0001

so here is a simple table:

0011 0000 = 0
0011 0001 = 1
0011 0010 = 2
0011 0011 = 3
0011 0100 = 4
0011 0101 = 5
0011 0110 = 6
0011 0111 = 7
0011 1000 = 8
0011 1001 = 9
