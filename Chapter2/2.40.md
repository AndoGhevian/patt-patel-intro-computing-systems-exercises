### 2.40 Write the decimal equivalents for these IEEE floating point numbers.
1. a. 0 10000000 00000000000000000000000
1. b. 1 10000011 00010000000000000000000
1. c. 0 11111111 00000000000000000000000
1. d. 1 10000000 10010000000000000000000

#### Solution

#### a. 0 10000000 00000000000000000000000

0 => + sign

10000000 - 127 = 1 exponent

00000000000000000000000 = fraction part 0 right to binary point

= +2^1 * 1.0 = +2

#### b. 1 10000011 00010000000000000000000

1 => - sign

10000011 excess code => subtract 127 => 4 exponent

val = 1.00010000000000000000000 * 2^4 = 10001.0 = 17

prepend sign => -17

#### c. 0 11111111 00000000000000000000000

representations where exponent is all 1's are used
to represent special things, like:

when there is fraction is all 0's then it is INFINITY

with sign bit it represents -INFINITY

#### d. 1 10000000 10010000000000000000000

1 sign bit => - sign minus

10000000 excess code = exponent of 1

it is normalized value because exponent code is not all 0's.

1.10010000000000000000000 * 2^1 = 11.001 =

= 3 + 2^-3 = 3 + 1/8 = 3.125
